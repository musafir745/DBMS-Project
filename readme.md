WanderLust is a hotel booking management system developed as a DBMS Lab project using Node.js, Express.js, EJS and MySQL. The project allows administrators to manage hotels, rooms, customers, and bookings through a simple web interface.

The system performs all basic CRUD (Create, Read, Update, Delete) operations and uses MySQL as the database with the mysql2 package. It was built to demonstrate the practical application of database management concepts, including table relationships, primary keys, foreign keys, and data management.
