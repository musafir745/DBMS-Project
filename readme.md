# WanderLust: Hotel Booking DBMS Project

WanderLust is a Hotel Booking Management System developed as a DBMS Lab project using Node.js, Express.js, EJS, Bootstrap, and MySQL.

The project allows users to browse hotels, book rooms, and manage bookings through a web-based interface. It also includes separate Customer and Administrator panels, where customers can manage their own accounts and bookings, while administrators can manage hotels, rooms, users, and bookings.

## Technologies Used

- Node.js
- Express.js
- EJS
- Bootstrap
- MySQL
- mysql2
